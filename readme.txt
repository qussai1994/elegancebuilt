EleganceBuilt Website - Deployment Package
==========================================

HOW TO USE:
-----------
1. Extract this ZIP file to any folder
2. Open index.html in any web browser — the site works fully offline
3. All images are in the images/ folder — do NOT move or rename them

TO DEPLOY ONLINE (e.g. Netlify - Free):
----------------------------------------
1. Go to https://netlify.com and sign up free
2. Drag and drop the entire extracted folder onto the Netlify dashboard
3. Your site will be live in seconds with a free URL

TO DEPLOY ON ANY WEB HOST:
---------------------------
- Upload the index.html file AND the images/ folder to your web server
- Keep the folder structure exactly as-is
